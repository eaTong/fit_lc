# 重构"我的"页面 - 整合数据看板

## 需求概述

将"数据"Tab页的看板功能迁移到"我的"页面，把我的页面打造为用户的个人数据中心首页。

## 目标

1. 看板（Dashboard）作为"我的"页面最上方的默认内容
2. 为其他功能（训练历史、趋势分析、围度记录、计划）提供快速入口
3. 计划页面也整合到"我的"Tab中
4. 移除底部Tab栏的"数据"Tab（因为已整合到"我的"）

## 路由变更

### 旧路由
- `/history` - 数据Tab（包含4个子Tab：看板/历史/趋势/围度）
- `/profile` - 我的页面（简单）

### 新路由
- `/profile` - 我的页面（整合看板+功能入口）
- `/history` - 训练历史页面（直接显示，无需子Tab）
- `/trends` - 趋势分析页面
- `/measurements` - 围度记录页面

## 底部Tab栏变更

### 旧Tab栏
```
首页 | 数据 | 计划 | 知识 | 我的
```

### 新Tab栏
```
首页 | 历史 | 计划 | 知识 | 我的
```

移除"数据"Tab，因为看板已移至"我的"页面。
