# F-011: 行为锚点绑定设计

## Context

在用户完成训练后的 30 分钟内，是引导用户记录数据的最佳时机。通过设计"行为锚点"——即在特定情境下触发的提示或引导——可以有效提升记录转化率。

---

## 行为锚点设计

### 锚点触发时机

| 时机 | 触发条件 | 锚点形式 |
|------|----------|----------|
| T+0 训练刚结束 | AI 检测到训练完成消息 | 即时展示庆祝动画 + "是否保存？" |
| T+30min 健身后 | 用户活跃但未记录 | Dashboard 空状态引导 |
| T+24h 次日 | 用户未记录 | 推送提醒（依赖 F-009） |

### 锚点 A: 训练结束即时引导

**触发条件**: AI 回复中检测到用户描述了训练内容，但尚未保存到数据库。

**当前实现**: AI 通过 `saveWorkout` tool 自动保存，但用户可能不明确知道记录已保存。

**优化方案**:
1. AI 在成功保存后返回确认消息，包含保存的关键数据摘要
2. 添加"撤销"按钮，让用户知道可以修改
3. 如检测到首次训练，触发动画庆祝

**触发流程**:
```
用户: "今天胸肌训练做了卧推100kg"
  → AI 调用 saveWorkout tool
  → 保存成功，返回: "✅ 已记录：卧推 100kg × 5组。首次训练，恭喜！🎉"
  → 前端检测 totalWorkouts === 1，触发 FirstTimeCelebration
```

### 锚点 B: Dashboard 空状态引导

**触发条件**: 用户进入 Dashboard，但无训练记录（或累计 = 0）。

**表现形式**:
```
┌─────────────────────────────────────┐
│  📋 开始记录你的第一次训练           │
│                                     │
│  通过自然语言告诉 AI 你今天做了什么  │
│  例如："今天深蹲 80kg 做了 5 组"    │
│                                     │
│  [开始训练]                          │
└─────────────────────────────────────┘
```

**设计要点**:
- 正面鼓励文案，强调"第一次"的特殊性
- 提供具体示例，降低描述门槛
- CTA 按钮直接跳转 Chat 页面

### 锚点 C: 次日提醒（依赖 F-009 推送服务）

**触发条件**: 用户昨日有训练计划执行记录，但今日未进入应用。

**表现形式**: Web Push 推送
```
标题: "昨天练了，今天呢？💪"
正文: "你的胸肌训练记录已保存，继续保持！"
行动: "查看记录" → 跳转 Dashboard
```

**频率控制**: 通过 `TriggerEvent` 表限制每个用户每周最多 2 次。

---

## 用户操作路径

### 路径 1: 训练中即时记录

```
1. 用户开始训练
2. 训练间隙打开 Chat
3. 输入: "卧推 100kg 5组"
4. AI 保存 → 展示确认
5. （可选）追加: "再加一组"
6. AI 更新记录
7. 训练结束，查看 Dashboard
```

### 路径 2: 训练后补录

```
1. 健身后打开 App
2. Dashboard 显示空状态
3. 点击"开始训练"
4. 跳转到 Chat
5. 输入: "今天下午练了背"
6. AI 询问具体动作/重量
7. 用户补充信息
8. AI 保存完整记录
```

## 技术实现

### 当前状态

- AI `saveWorkout` tool 已自动保存数据
- AI 回复内容包含保存成功的确认消息
- `FirstTimeCelebration` 组件已集成到 Chat 页面
- Dashboard 有空状态引导

### 缺失部分

1. **锚点 B 的空状态 UI** - 需要在 Dashboard 添加引导卡片
2. **锚点 C 推送服务** - 需要 F-009 基础设施

### 建议: 锚点 B 快速实现

在 `Dashboard.tsx` 的最近训练区域，当 `recentWorkouts.length === 0` 时，添加引导卡片：

```tsx
{recentWorkouts.length === 0 && (
  <div className="text-center py-12 bg-primary-secondary border-2 border-dashed border-border">
    <p className="text-4xl mb-4">📋</p>
    <h3 className="font-heading text-xl mb-2">开始记录你的第一次训练</h3>
    <p className="text-text-secondary mb-4">
      通过自然语言告诉 AI 你今天做了什么
    </p>
    <code className="text-accent-orange">"今天深蹲 80kg 做了 5 组"</code>
    <div className="mt-4">
      <Link to="/chat" className="btn-primary">
        开始训练
      </Link>
    </div>
  </div>
)}
```

## 验证方案

2. 新用户第一次通过 Chat 保存训练后，应看到庆祝动画
3. 已有多次训练的用户看到正常 Dashboard
