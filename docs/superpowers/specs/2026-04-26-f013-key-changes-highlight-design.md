**状态：** 设计完成

## 一、概述

**功能名称：** 重点变化Highlight
**功能描述：** 在趋势页展示本周/本月最重要的进步和退步，使用箭头和颜色直观标识，帮助用户快速了解关键变化。

**优先级：** P0（L1 基础功能层）
**依赖关系：** F-012 智能总结的展示载体

---

## 二、展示位置

**趋势页布局：**
```
┌─────────────────────────────────────────────┐
│  📊 AI 智能总结                              │
│  本周训练4次，深蹲重量提升至100kg...         │
│  [查看详情 ▼]                               │
├─────────────────────────────────────────────┤
│  🔥 重点变化                                │
│                                             │
│  ↑ 深蹲重量    95kg → 100kg    +5kg        │
│  ↑ 卧推重量    80kg → 85kg    +5kg         │

```

## 三、Highlight 内容

### 3.1 训练维度

| 指标 | 变化示例 | 说明 |
|------|----------|------|
| 重量 | 深蹲 95kg → 100kg (+5kg) | 单次最大重量变化 |
| 组数 | 引体向上 3组 → 5组 (+2组) | 训练组数变化 |
| 训练量 | 本周累计 15,000kg → 18,000kg | 总训练量变化 |
| 训练次数 | 3次 → 4次 (+1次) | 周训练频次变化 |

### 3.2 围度维度

| 胸围 | 96cm → 98cm (+2cm) | 胸部围度变化 |

| 臀围 | 95cm → 96cm (+1cm) | 臀部围度变化 |
| 臂围 | 32cm → 33cm (+1cm) | 臂部围度变化 |

## 四、视觉样式

### 4.1 进步标识（绿色）

```
↑ 深蹲重量    95kg → 100kg    [+5kg]
```

- 箭头：绿色 ↑
- 数值：新值 + 变化量
- 整体：绿色主题

### 4.2 退步标识（红色）

```
↓ 体重        75kg → 74kg    [-1kg]
```

- 箭头：红色 ↓
- 数值：新值 + 变化量
- 整体：红色主题

### 4.3 无变化

```
→ 体重        75kg → 75kg    [持平]
```

- 灰色中性展示

---

## 五、筛选规则

### 5.1 显示条件

- 变化幅度超过阈值（默认 ±5% 或绝对值 ±0.5kg/cm）
- 排除微小变化（噪音过滤）

### 5.2 排序规则

1. 进步优先（正向激励）
2. 变化幅度大的优先
3. 训练指标优先于围度指标

### 5.3 数量限制

- 最多显示 5-7 条重点变化
- 避免信息过载

## 六、技术实现

### 6.1 数据来源

- 训练记录（workouts, workout_exercises）
- 围度记录（body_measurements, measurement_items）
- 历史对比数据

### 6.2 计算逻辑

```javascript
// 计算变化
const change = currentValue - previousValue;
const changePercent = (change / previousValue) * 100;

// 判断是否显著变化
const isSignificant = Math.abs(changePercent) >= 5 || Math.abs(change) >= 0.5;
```

### 6.3 组件设计

```typescript
// components/KeyChangesHighlight.tsx
interface KeyChangesHighlightProps {
  period: 'week' | 'month';
  changes: ChangeItem[];
}

interface ChangeItem {
  type: 'workout' | 'measurement';
  name: string;
  previousValue: number;
  currentValue: number;
  unit: string;
  direction: 'up' | 'down' | 'same';
}
```
