## 运行命令

```bash
# 后端测试
cd backend && npm test

# 前端单元测试
cd frontend && npm test

# 前端 E2E 测试
cd frontend && npx playwright test

# 运行特定 E2E 测试
cd frontend && npx playwright test --grep "@auth"
cd frontend && npx playwright test --grep "@chat"
cd frontend && npx playwright test --grep "@admin"
```
